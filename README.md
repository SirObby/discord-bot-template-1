# MatthewTGMs Discord Bot Template
## How To Setup
1. Clone the repository to your desired location
2. Rename the folder from `discord-bot-template` to whatever you wish
3. Go to (this page)[https://discord.com/developers/applications] and create a new application
4. Create a bot using build-a-bot on the bot section
5. Copy your bot token
6. In the configurations [`conf/config.json`] paste your token in the `token` field and edit the prefix and version to your liking
7. You can now head on to create your own commands!
